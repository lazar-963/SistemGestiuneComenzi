# Proiect Medii si Platforme de Dezvoltare 2026

## Design Patterns folosite:
1. **Strategy Pattern**: Folosit pentru calcularea prețului final în funcție de tipul de client (VIP/Regular), respectând Open-Closed Principle.
2. **Observer Pattern**: Folosit pentru notificarea sistemelor de Email/SMS atunci când starea unei comenzi se schimbă.

## Testare:
Proiectul conține teste unitare realizate cu NUnit pentru verificarea ambelor pattern-uri.
